## Цель
Увидеть дизайн дашборда ДО загрузки данных из Excel таблиц.

---

## 🚀 Быстрый запуск (1 команда):

```bash

```

Этот скрипт:
1. ✅ Запустит миграцию базы данных
2. ✅ Создаст 3 самолета с нулевыми данными
3. ✅ Запустит сервер
4. ✅ Откроет браузер

## 💡 После оценки визуальной части:

### Когда будете готовы вводить реальные данные:

1. **Установка двигателя:**
   - Перейти в `Installation`
   - Выбрать двигатель, самолет, позицию
   - Ввести TT и TC (можно 0 если новый)
   - Сохранить

2. **Добавление налета:**
   - Перейти в `Utilization (ATLB)`
   - Выбрать самолет
   - Ввести время полета и циклы
   - Система автоматически обновит все счетчики

3. **Параметры двигателя:**
   - Перейти в `ENG Parameters`
   - Ввести N1/N2 для cruise и takeoff
   - Данные отобразятся в карточках двигателей

## ❓ Часто задаваемые вопросы:

**Q: Почему все нули?**
A: Это специально! Сейчас вы смотрите только дизайн. Данные появятся когда начнете их вводить.

**Q: Можно ли изменить названия самолетов?**
A: Да, в базе данных. Или просто переименуйте при вводе реальных данных.

**Q: Как будут выглядеть карточки с данными?**
A: Запустите `python add_test_data.py` чтобы увидеть с примерами данных (но это перезапишет ваши будущие данные, так что лучше потом удалить базу и начать с нуля).

**Q: Безопасно ли это для моих будущих данных?**
A: Да! Это создает только базовую структуру. Когда будете вводить реальные данные - они просто обновят значения с 0 на реальные цифры.
