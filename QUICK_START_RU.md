## 🔥 Главная фишка

**Автоматический расчет налета:**

1. Установили двигатель → система запомнила TSN/CSN
2. Добавили данные из бортжурнала (ATLB/Utilization)
3. **Система автоматически** посчитала сколько налетал этот двигатель на этом самолете!

**Пример:**
```
Двигатель 880123:
- Был установлен с TSN = 15,000 hrs
- Самолет налетал еще 1,500 hrs
- Дашборд показывает:
  ✅ Total TSN: 16,500 hrs (общий налет)
  ✅ TSN on A/C: 1,500 hrs (налет на этом самолете) ← ВАЖНО!
```

## ✨ Как использовать:

### Установить двигатель:
1. Installation → выбрать двигатель, самолет, позицию
2. Ввести TT и TC (даже если 0 после ремонта)
3. Сохранить

### Добавить полет:
1. Utilization (ATLB) → выбрать самолет
2. Ввести время полета и циклы
3. Сохранить
4. **Все счетчики обновятся автоматически!**

### Посмотреть данные:
1. Главный экран → кликнуть ▼ под карточкой самолета
2. Увидите все 4 позиции с подробными данными
