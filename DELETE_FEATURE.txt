================================================================
  ФУНКЦИЯ УДАЛЕНИЯ ЗАПИСЕЙ - ИНСТРУКЦИЯ
================================================================

✅ ДОБАВЛЕНО:
- Кнопки удаления (корзины) для всех записей
- Подтверждение удаления с предупреждением
- API endpoint для удаления двигателей
- Защита от случайного удаления

================================================================
  КАК ПОЛЬЗОВАТЬСЯ
================================================================

1. MASTER ENGINE LIST (Список двигателей):
   
   Шаг 1: Откройте Master Engine List → вкладка "Список двигателей"
   Шаг 2: Нажмите кнопку "Edit Table" (желтая кнопка)
   Шаг 3: Слева от каждой строки появятся красные корзины
   Шаг 4: Кликните на корзину рядом с нужным двигателем
   Шаг 5: Появится окно подтверждения:
           ⚠️ ВНИМАНИЕ!
           Вы действительно хотите удалить двигатель?
           Serial Number: 123456
           
           [OK]    [Cancel]
   
   Шаг 6: Нажмите OK для удаления или Cancel для отмены
   Шаг 7: После удаления таблица автоматически обновится
   Шаг 8: Нажмите "Save Changes" или "Cancel" для выхода

2. INSTALLATION HISTORY (История установок):
   
   Шаг 1: Откройте Installation → вкладка "Вся информация"
   Шаг 2: Нажмите кнопку "Edit" (желтая)
   Шаг 3: Появятся кнопки удаления
   Шаг 4: Кликните на корзину → подтвердите удаление
   Шаг 5: Нажмите "Done" для выхода

================================================================
  ЗАЩИТА ОТ ОШИБОК
================================================================

❌ НЕЛЬЗЯ удалить двигатель, который установлен на самолет
   Сначала снимите его через Remove Action

❌ НЕЛЬЗЯ отменить удаление после подтверждения
   Будьте внимательны!

✅ МОЖНО удалять только двигатели со статусом:
   - SV (Serviceable)
   - US (Unserviceable)
   - REMOVED (Снятый)

================================================================
  ЧТО ПРОИСХОДИТ ПРИ УДАЛЕНИИ
================================================================

1. Запись удаляется из базы данных
2. Счетчики на Dashboard обновляются
3. Таблица перезагружается
4. История операций сохраняется (логи остаются)

================================================================
  БЫСТРЫЙ ТЕСТ
================================================================

1. Добавьте тестовый двигатель:
   - ESN: TEST123
   - Location: SHJ

2. Перейдите в Master Engine List
3. Нажмите "Edit Table"
4. Увидите красную корзину слева
5. Кликните → подтвердите → двигатель удален
