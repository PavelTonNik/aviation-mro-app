================================================================
  БЫСТРАЯ ИНСТРУКЦИЯ ПО ЗАПУСКУ
================================================================

1. ЗАПУСК СЕРВЕРА:
   - Дважды кликните по файлу: START.bat
   - Откроется окно командной строки с сервером
   - Дождитесь сообщения: "Application startup complete"

2. ОТКРОЙТЕ БРАУЗЕР:
   - Перейдите по адресу: http://127.0.0.1:8000
   - Нажмите Ctrl+Shift+R (жесткая перезагрузка, очистка кеша)

3. ДОБАВЛЕНИЕ ДВИГАТЕЛЯ С ЛОКАЦИЕЙ:
   - Откройте меню слева: Master Engine List
   - Перейдите на вкладку: "Добавить двигатель"
   - Заполните поля:
     ✓ Model: CF6-80C2
     ✓ Orig. ESN: 123456 (обязательно)
     ✓ Status: SV (Serviceable)
     ✓ Current Location: SHJ - Sharjah (ОБЯЗАТЕЛЬНО!)
     ✓ TT и TC: оставьте 0
   - Нажмите "Save Record"

4. ПРОВЕРКА:
   - Вернитесь на Dashboard (кнопка в меню слева)
   - Кнопка "SHJ (Sharjah)" должна показать: 1
   - Кликните на кнопку → откроется таблица с двигателем

================================================================
  ЕСЛИ НЕ ВИДНО ПОЛЕ "CURRENT LOCATION"
================================================================

1. Остановите сервер (закройте окно CMD)
2. Нажмите Ctrl+F5 в браузере (жесткая перезагрузка)
3. Запустите сервер снова (START.bat)
4. Обновите страницу еще раз
